## Block elements

Block components create block level containers:

<div>
  <Block border>Block content</Block>
  <Block border>Block content</Block>
</div>

## Inline block elements

Pass A truthy `inline` prop to create an block element with `display: inline-block`:

<div>
  <Block inline border>
    Block content
  </Block>
  <Block inline border>
    Block content
  </Block>
</div>
