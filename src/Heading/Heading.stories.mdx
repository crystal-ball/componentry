## Displays

The `display` variant will set the display styles:

<Heading display={1}>Fancy!</Heading>
<Heading display={2}>Fancy!</Heading>

## Headings

The `as` prop is used to create a `.heading-{1,4}` class:

<Heading as='h1'>Componentry stories</Heading>
<Heading as='h2'>Componentry stories</Heading>
<Heading as='h3'>Componentry stories</Heading>
<Heading as='h4'>Componentry stories</Heading>

## Subheadings

The `subheading` variant will set subheading styles using the `as` prop

<Heading as='h2' variant='subheading'>
  Componentry stories
</Heading>
<Heading as='h3' variant='subheading'>
  Componentry stories
</Heading>
<Heading as='h4' variant='subheading'>
  Componentry stories
</Heading>
