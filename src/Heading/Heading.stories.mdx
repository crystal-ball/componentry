Use the `as` prop to set the heading level:

<Heading as='h1'>Componentry stories</Heading>
<Heading as='h2'>Componentry stories</Heading>
<Heading as='h3'>Componentry stories</Heading>
<Heading as='h4'>Componentry stories</Heading>

## Display Headings

The `display` prop will set the display styles:

<Heading display={1}>Fancy!</Heading>
<Heading display={2}>Fancy!</Heading>

## Subheadings

The `.subheading` class can be used to create heading levels:

<Heading>
  Componentry <span className='subheading'>components</span>
</Heading>
<Heading as='h2'>
  Componentry <span className='subheading'>components</span>
</Heading>
<Heading as='h3'>
  Componentry <span className='subheading'>components</span>
</Heading>
<Heading as='h4'>
  Componentry <span className='subheading'>components</span>
</Heading>
