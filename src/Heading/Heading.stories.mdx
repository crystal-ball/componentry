## Displays

The `display` variant will set the display styles:

<Heading variant='display-1'>Fancy!</Heading>
<Heading variant='display-2'>Fancy!</Heading>

## Headings

The `as` prop is used to create a `.heading-{1,4}` class:

<Heading variant='heading-1'>Componentry stories</Heading>
<Heading variant='heading-2'>Componentry stories</Heading>
<Heading variant='heading-3'>Componentry stories</Heading>
<Heading variant='heading-4'>Componentry stories</Heading>
<Heading variant='heading-5'>Componentry stories</Heading>

## Subheadings

The `subheading` variant will set subheading styles using the `as` prop

<Heading variant='subheading-1'>Componentry stories</Heading>
<Heading variant='subheading-2'>Componentry stories</Heading>
<Heading variant='subheading-3'>Componentry stories</Heading>
