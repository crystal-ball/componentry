# Grid

Grid is a convenience component for creating `display: grid` containers.

## Block elements

Grid components create block level containers:

<Grid className='grid-cols-2' gap={4}>
  <div className='border'>Grid item</div>
  <div className='border'>Grid item</div>
</Grid>

## Inline block elements

Pass A truthy `inline` prop to create an element with `display: inline-grid`:

<Grid inline className='grid-cols-2' gap={4}>
  <div className='border'>Grid item</div>
  <div className='border'>Grid item</div>
</Grid>
