# Alert

Alerts can contain any HTML elements, and can be dismissible or static. A dismissible
Alert requires Componentry Active `active` and `deactivate` props, which can be passed
directly or the Alert can be nested inside of an Active component to automatically create
an uncontrolled Alert.

<Canvas>
  <Story name='Alert'>
    <Alert color='primary'>
      <Text variant='h3' className='C9Y-AlertHeading'>
        Well done!
      </Text>
      <Text>You successfully read this important alert message.</Text>
      <hr className='my-2' />
      <Link href='#demo' className='C9Y-AlertLink'>
        Go home
      </Link>
    </Alert>
  </Story>
</Canvas>

Styles are also included for: `alert-heading`, `alert-link`, and `<hr />` elements.

## Dismissible

Pass `dismissible` to an Alert to show a close icon which will call the `deactivate` prop
on click.
