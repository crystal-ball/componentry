# Flex

Flex is a convenience component for creating `display: flex` containers.

<ArgsTable of={Flex} />

## Block containers

Flex components create block level flex containers:

<Flex>
  {[1, 2, 3].map((key) => (
    <div key={key} className='border'>
      Flex content #{key}
    </div>
  ))}
</Flex>

## Inline containers

Create inline level flex containers using the `inline` prop:

<Flex inline>
  {[1, 2].map((key) => (
    <div key={key} className='border'>
      Flex content #{key}
    </div>
  ))}
</Flex>
<Flex inline>
  {[1, 2].map((key) => (
    <div key={key} className='border'>
      Flex content #{key}
    </div>
  ))}
</Flex>

## Flex direction

Override the default direction of `row` with the `direction` prop:

<Flex direction='row-reverse'>
  {[1, 2, 3].map((key) => (
    <div key={key} className='border'>
      Flex content #{key}
    </div>
  ))}
</Flex>

## Justify and align

Set the flex items justification and alignment with the `justify` and `align` props:

<Flex justify='center' align='center'>
  {[1, 2, 3].map((key) => (
    <div key={key} className='border'>
      Flex content #{key}
    </div>
  ))}
</Flex>

## Flex wrap

Override the flex wrap behavior with the `wrap` props:

<Flex wrap='wrap-reverse'>
  {Array(12)
    .fill(0)
    .map((_, key) => (
      // eslint-disable-next-line react/no-array-index-key
      <div key={key} className='border'>
        Flex content #{key}
      </div>
    ))}
</Flex>
