## Variants

A **primary** variant button:

<Button base='btn' variant='primary' onClick={action('button click')}>
  Button
</Button>

A **secondary** variant button:

<Button variant='secondary' onClick={action('button click')}>
  Button
</Button>

---

## Sizes

Pass a `size` to set a button size:

<div className='d-flex'>
  <div className='btn-container-x'>
    <Button size='sm'>Button</Button>
    <Button>Button</Button>
    <Button size='lg'>Button</Button>
  </div>
</div>

## Full width buttons

Pass `width="100%"` to make a button fill its parent container:

<Button width='100%'>Full width button</Button>

## Disabled

Pass `disabled` to make a button disabled:

<div className='btn-container-x'>
  <Button variant='primary' disabled>
    Button
  </Button>
  <Button variant='secondary' disabled>
    Button
  </Button>
</div>

## Buttons with routing

A Button with routing handlers:

<Button href='#test'>Button</Button>

## Input buttons

Use the `as` prop to render input elements styled as buttons:

<div className='d-flex justify-content-center btn-container-x'>
  <Button>Button</Button>
  <Button as='input' value='Input' />
  <Button as='input' type='submit' value='Submit' />
  <Button as='input' type='reset' value='Reset' />
</div>
