## Sizes

Pass a `size` to set a button size:

<Canvas>
  <Story
    name='Sizes'
    decorators={[
      (Story) => (
        <div className='flex flex-row items-center gap-4'>
          <Story />
        </div>
      ),
    ]}
  >
    <Button size='sm'>Button</Button>
    <Button>Button</Button>
    <Button size='lg'>Button</Button>
  </Story>
</Canvas>

## Full width buttons

Pass `fullWidth` to make a button fill its parent container:

<Button fullWidth>Button</Button>

## Disabled

Pass `disabled` to make a button disabled:

<div className='flex flex-row items-center gap-4'>
  <Button variant='primary' disabled>
    Button
  </Button>
  <Button variant='secondary' disabled>
    Button
  </Button>
</div>

## Buttons with hrefs

Buttons with an `href` will be rendered as anchors:

<Button href='#test'>Button</Button>

## Input buttons

Pass an `as` prop to render input elements styled as buttons:

<div className='flex flex-row items-center gap-4'>
  <Button>Button</Button>
  <Button as='input' value='Input' />
  <Button as='input' type='submit' value='Submit' />
  <Button as='input' type='reset' value='Reset' />
</div>
