A text variant button:

<Button variant='text' onClick={action('button click')}>

## Theme colors

Pass a `color` to set a theme color background:

<Flex>
  {['primary', 'success', 'warning', 'error'].map((color) => (
    <Button key={color} variant='primary' color={color} onClick={action('button click')}>
      Button
    </Button>
  ))}
</Flex>

<Flex mt='md'>
  {['primary', 'success', 'warning', 'error'].map((color) => (
    <Button
      key={color}
      variant='secondary'
      color={color}
      onClick={action('button click')}
    >
      Button
    </Button>
  ))}
</Flex>

<Flex mt='md'>
  {['primary', 'success', 'warning', 'error'].map((color) => (
    <Button key={color} variant='text' color={color} onClick={action('button click')}>
      Button
    </Button>
  ))}
</Flex>

<Flex mt='md'>
  {['primary', 'success', 'warning', 'error'].map((color) => (
    <Button key={color} variant='anchor' color={color} onClick={action('button click')}>
      Button
    </Button>
  ))}
</Flex>

## Sizes

Pass a `size` to set a button size:

<div className='d-flex justify-content-center '>
  <div className='btn-container-x'>
    <Button size='sm'>Button</Button>
    <Button>Button</Button>
    <Button size='lg'>Button</Button>
  </div>
</div>

## Full width buttons

Pass `width="100%"` to make a button fill its parent container:

<Button width='100%'>Full width button</Button>

## Disabled

Pass `disabled` to make a button disabled:

<Button variant='primary' disabled>
  Button
</Button>

<Button variant='secondary' disabled>
  Button
</Button>

<Button variant='text' disabled>
  Button
</Button>

<Button variant='anchor' disabled>
  Button
</Button>

## Input buttons

Use the `as` prop to render input elements styled as buttons:

<div className='d-flex justify-content-center btn-container-x'>
  <Button>Button</Button>
  <Button as='input' value='Input' />
  <Button as='input' type='submit' value='Submit' />
  <Button as='input' type='reset' value='Reset' />
</div>
