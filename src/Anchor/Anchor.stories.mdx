# Anchors

Anchor element with an href:

<Anchor href='#'>Anchor element</Anchor>

<Props of={Anchor} />

## Theme colors

Pass a `color` to set a theme color background:

<div className='d-flex justify-content-center flex-column'>
  {colors.map(color => (
    <Anchor color={color} href='#' key={color}>
      Achor element
    </Anchor>
  ))}
</div>

## Disabled

Pass `disabled` to make an anchor disabled:

<Anchor disabled>Anchor element</Anchor>

## Button class

Create a button element with anchor styles by passing `as='button'` to an Anchor:

<div className='d-flex justify-content-around btn-container-x'>
  <Anchor href='#'>Anchor element</Anchor>
  <Anchor as='button' onClick={action('button anchor')}>
    Button element
  </Anchor>
</div>

</ElementsProvider>
