Anchor components render an anchor tag with anchor styles by default:

<Anchor href='#'>Anchor element</Anchor>

## Theme colors

Pass a `color` prop to theme the anchor font color:

<div className='d-flex flex-column'>
  {colors.map((color) => (
    <Anchor color={color} href='#' key={color}>
      Achor element
    </Anchor>
  ))}
</div>

---

## Disabled

Pass `disabled` to make an anchor disabled:

<Anchor disabled>Anchor element</Anchor>

---

## Button variant

Create an anchor with buttons style by passing `variant="btn"` to an Anchor:

<Anchor variant='btn' color='danger' outline href='#'>
  Go Back
</Anchor>
<Anchor variant='btn' color='anchor' href='#'>
  Anchor color button
</Anchor>
