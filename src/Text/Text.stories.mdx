## Body variant

The default `variant` for Text components is `body`:

<Text>This is a body paragraph. Use it for primary application text content.</Text>

## Lead variant

The `lead` variant will render a lead style paragraph:

<Text variant='lead'>This is a lead paragraph that will stand out.</Text>

## Caption variant

The `caption` variant will render a caption style paragraph:

<Text variant='caption'>This is a caption paragraph for details</Text>

## Block (paragraph) text

Paragraph elements are rendered by default:

<Text>Componentry default text element</Text>

## Inline (span) text

A truthy `inline` prop renders a `span` element:

This is a larger block of text that has a <Text fontWeight='bold' inline>bold</Text> text
span inside of it.

## Sizes

The `size` prop sets a font size:

<Text size='xl'>Example extra large text content.</Text>
<Text size='lg'>Example large text content.</Text>
<Text size='base'>Example text content.</Text>
<Text size='sm'>Example small text content.</Text>
<Text size='xs'>Example extra small text content.</Text>
