## Space organization

Pass the library size variants to one of the library spacing props to set margin or
padding.

<div>
  {['xs', 'sm', 'md', 'lg', 'xl'].map((space) => (
    <Block key={space}>
      <small className='font-family-monospace'>{space}</small>:{' '}
      <Block backgroundColor='demo' pr={space} height='8px' inline />
    </Block>
  ))}
</div>

- Margin props: `m`, `mt`, `mr`, `mb`, `ml`, `mx`, `my`
- Padding props: `p`, `pt`, `pr`, `pb`, `pl`, `px`, `py`
- Size variants: `xs`, `sm`, `md`, `lg`, `xl`

## Margin auto

Pass `'auto'` to any margin prop and it will be applied as a style.

eg `mx='auto'`

<div className='w-100'>
  <Block background='demo' height={16} width={16} mx='auto' />
</div>
