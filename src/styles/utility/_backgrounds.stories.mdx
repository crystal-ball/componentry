### Background colors

Set the background color to any color added to the `theme.palette.background-colors` map.

<div className='d-flex justify-content-center'>
  {['demo'].map((color) => (
    <div key={color}>
      <Block
        width={100}
        height={100}
        background={color}
        mx='sm'
        className='radius border'
      />
      <small className='small ml-2'>.background-{color}</small>
    </div>
  ))}
</div>

### Transparent

Add `.background-transparent` to override the background to be transparent.
