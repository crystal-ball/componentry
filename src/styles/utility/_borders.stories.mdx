### Border & border directions

Add a border using the border utility classes.

<div className='d-flex justify-content-center'>
  <div>
    <Block width={100} height={15} className={`border mx-2`} background='100' />
    <small className='small ml-2'>.border</small>
  </div>
  {['top', 'right', 'bottom', 'left'].map((border) => (
    <div key={border}>
      <Block
        width={120}
        height={15}
        mx='sm'
        className={`border-${border}`}
        background='100'
      />
      <small className='small ml-2'>.border-{border}</small>
    </div>
  ))}
</div>

### Border colors

Set the color of a border to any theme color as well as any color added to the
`$border-colors` map.

<div className='d-flex justify-content-center'>
  {['demo'].map((color) => (
    <div key={color}>
      <Block
        width={120}
        height={15}
        border
        borderColor={color}
        mx='sm'
        background='100'
      />
      <small className='small ml-2'>.border-{color}</small>
    </div>
  ))}
</div>

### Border Width

Increase the width of a border using the `lg` and `xl` border width classes.

<div className='d-flex justify-content-center'>
  <div>
    <Block width={120} height={15} border borderWidth='lg' mx='sm' />
    <small className='small ml-2'>.border-width-lg</small>
  </div>
  <div>
    <Block width={120} height={15} border borderWidth='xl' mx='sm' />
    <small className='small ml-2'>.border-width-xl</small>
  </div>
</div>

### Border radius directions

Add a directional border radius.

<div className='d-flex justify-content-center'>
  {['top', 'right', 'bottom', 'left'].map((className) => (
    <div key={className}>
      <Block
        width={100}
        height={100}
        background='100'
        mx='base'
        className={`border border-radius-${className}`}
      />
      <small className='small ml-sm'>.border-radius-{className}</small>
    </div>
  ))}
</div>

#### Radius circle

Use `.border-radius-circle` to create circles or pills by adjusting the width and height
ratio of an element

      width={50}
      height={50}
      border
      className='border-radius-circle'
      mx='sm'
      background='100'
    />
    <small className='small ml-sm'>.border-radius-circle</small>
