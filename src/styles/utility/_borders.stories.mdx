## Borders

### Border base

<div className='d-flex justify-content-center'>
  <div>
    <Block width={100} height={15} className={`border mx-2`} background='100' />
    <small className='small ml-2'>.border</small>
  </div>
  {['top', 'right', 'bottom', 'left'].map(border => (
    <div key={border}>
      <Block
        width={120}
        height={15}
        mx='sm'

  ))}
</div>

### Border radius

! Want radius levels, eg .radius-sm, .radius, .radius-lg

1. radius -> add a border-radius, accept sm, lg to change radius size
1. radiusTop, radiusRight, etc. -> same but for one side
1. circle
1. pill

<div className='d-flex justify-content-center'>

  </div>

The `.radius-circle` class can be used to create circles or pills by adjusting the width
and height ratio of an element
