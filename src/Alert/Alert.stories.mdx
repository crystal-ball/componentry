# Alerts

<Description
  of={Alert}
  markdown='Provide contextual feedback messages for user actions using theme colors.'
></Description>

Alerts can contain any HTML elements, and can be dismissible or static. A dismissible
Alert requires Componentry Active `active` and `deactivate` props, which can be passed
directly or the Alert can be nested inside of an Active component to automatically create
an uncontrolled Alert.

<Alert color='primary' outline={boolean('outline', false)}>
  <h4 className='alert-heading'>Well done!</h4>
  <p className='mb-0'>You successfully read this important alert message.</p>
  <hr className='hr' />
  <a href='#' className='a alert-link'>
    Go home
  </a>
</Alert>

<Props of={Alert} />

You can also create outline alerts that have only a border-left

<Alert color='primary' outline>
  <h4 className='alert-heading'>Well done!</h4>
  <p className='mb-0'>You successfully read this important alert message.</p>
  <hr className='hr' />
  <a href='#' className='a alert-link'>
    Go home
  </a>
</Alert>

## Dismissible

Create a dismissible alert by wrapping it in an `Active` component.

<Active defaultActive>
  <Alert color='primary' dismissible>
    <h4 className='alert-heading'>Well done!</h4>
    <p className='mb-0'>You successfully read this important alert message.</p>
    <hr className='hr' />
    <Anchor href='#' className='alert-link'>
      Go home
    </Anchor>.
  </Alert>
</Active>

### Aria title

Provide an aria title for alerts with better accessibility.

<Alert color='primary' ariaTitle='Alert provides context on setting the aria title'>
  Pass an `ariaTitle` to provide more context.
</Alert>
